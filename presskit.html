PressKit

For Journalists and Bloggers.

Description

Maxel is a download manager and accelerator designed to fit into your workflow on the Mac.

Highlights

  • Splits downloads in up to 100 connections, increasing speed (how it works)
  • Saves progress files so you can resume downloads even between restarts
  • Many ways to add files, from dragging & dropping snippets of text with links to browser extensions for Chrome, Safari, and Firefox
  • Access authenticated downloads (sites which need login) with a built-in browser
  • Detect stalled downloads and automatically retry them
  • Compute MD5/SHA hashes of files to detect corruption
  • Supports HTTP, HTTPS, SFTP, and FTP
  • Supports public key authentication for SFTP
