FAQ

Here's where I can answer common questions. If you have a question that's not on this list, tweet @maxelapp or email support@maxelapp.com and I'll be glad to help!

How do I add a download?

There are many ways to initiate new downloads:

1. Drag & Drop

The easiest is to highlight text containing links in your browser (like Safari) and then drag this text into Maxel's dock icon or window.

2. Using the Ad Sheet

You could also press Command + N (âŒ˜N) in Maxel to open the Add Sheet. Click for more details on the Add Sheet.

3. Extensions for your Browser

Many users use one of Maxel's browser extensions, which provide even easier ways to add downloads.

4. Using the Built-in Browser

Downloads which require a login to access can be started from Maxel's Built-in Browser. Or, you can use the built-in browser if you have many downloads you want to start. Any downloads initiated from the built-in browser are automatically downloaded through Maxel.

Maxel says my download doesn't support multipart connections
Not all servers support multipart downloading. Some services restrict this capability to paid memberships, or may disable it entirely to save bandwidth. In this case, Maxel will revert back to a normal, single connection download which isn't resumable. If you have a download which Maxel won't split which it should, email me with a link and I'll take a look: support@maxelapp.com
I'm not seeing my download speed improve?
Maxel will only increase your speeds up to the bandwidth of your computer's connection. So if you have a 3 Mbps cable connection, that's the fastest you'll go. But in this case, Maxel will help if the server you're downloading from will only give you 300 kbps—just crank up the threads to 10, and you should get about 3 Mbps.
Maxel gets slow when I use 100+ connections?
At this point you're not gaining speed due to the overhead of establishing each connection. Try reducing the number of connections to 10 and work your way up from there.
Why does my SFTP download fail with > 10 connections?
The error message is: "Failure establishing SSH session. You may have too many connections..." The SSH server probably defaults to a maximum of 10 connections. To increase it, modify sshd_config (usually /etc/sshd/sshd_config) and delete the line if it exists:
MaxStartups 10:30:60

Add the line
MaxStartups 100:30:160

This will allow you to establish up to 100 connections simultaneously.
How can I add downloads programatically?

Maxel responds to the maxel:// protocol. See API help for details.

Maxel doesn't open when I intercept or initiate a download from a browser extension

Sometimes LaunchServices messes up and fails to target Maxel. This can happen if you upgraded from the demo and kept both apps, or have multiple copies of Maxel on your computer.

First, make sure to remove the Demo if you still have it, and make sure Maxel.app is installed in your Applications folder.

If that doesn't work, try resetting the LaunchServices database. In 10.9:

/System/Library/Frameworks/CoreServices.framework/Frameworks/LaunchServices.framework/Support/lsregister -kill -r -domain local -domain system -domain user
