Set it & Forget it

Maxel manages your downloads for you, so you don't have to. Add as many files as you want, and Maxel will download them without your intervention.

Maxel Saves

Maxel saves partial downloads so you can resume them anytime, and you won't lose your progress if your computer restarts.

Downright Delightful

An experience only possible in a native app.

Because Maxel is a Mac app, it behaves as you would expect. Keyboard shortcuts, Quicklook, and more—Maxel will make you feel right at home.

Other features include:

  • Drag & drop snippets of webpages into Maxel to download any links within
  • Download files from Maxel's built-in browser, which handles authenticated downloads with ease
  • Kickoff downloads from within your own browser
  • Computing MD5/SHA1 hashes of files to detect corruption
  • SSH public key authentication for SFTP
  • API for adding downloads in scripts
  • Supports the most common data transfer protocols: http, https, sftp, ftp.
