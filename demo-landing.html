Hello there!

I first started this project two years ago as a personal tool, since I was dissatisfied with the existing download accelerators for Mac OS X. I wanted something native, fast, and fun to use.

By using this demo, you've gotten a glimpse of what Maxel can do.

But development won't stop here. With your feedback, I have a roadmap for an exciting v1.1, which will have features like Automatic download mode (no more need to specify a connection count).
